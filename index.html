1. Task Inventory 검색 및 탐색

본 도구에는 O*NET SOC 코드, OSHA 위험 등급, 산업 표준(IATF, VDA, CQI, AIAG, AS9100, IEC 등)에 매핑된 191개의 Task가 포함되어 있습니다. 미국 남동부 에너지벨트(AL, TN, GA, SC, NC) 지역의 6개 산업을 다룹니다: 자동차(43), EV 배터리(31), 태양광(18), 항공우주(32), 일반 제조(53), 원자력(14).

검색바로 키워드, SOC 코드, 표준명 검색이 가능합니다. 각 카드의 ▶ Details를 누르면 상세 설명, 임금, 교육시간, 적용 표준, KSA 점수가 펼쳐집니다.

2. 복수선택 필터

Industry와 Department 모두 여러 개를 동시에 선택할 수 있습니다. 클릭하면 선택/해제가 토글되며, All을 누르면 전체 초기화됩니다. 예: Automotive + EV Battery를 동시에 선택하면 두 산업의 Task만 표시됩니다.

3. JD Matcher (직무기술서 매칭)

상단 JD Matcher 영역에 채용공고나 직무기술서를 텍스트로 붙여넣기하거나 키워드를 직접 입력하면 자동으로 키워드를 추출하고, 191개 Task와 관련성 점수를 산출합니다. 상위 25개 매칭 결과가 매칭 키워드와 함께 표시됩니다.

4. Cart & Analysis (장바구니 & 분석)

각 Task 카드의 + Add 버튼으로 Cart에 담을 수 있습니다. Cart에 Task가 담기면 상단에 Analysis 패널이 자동으로 표시됩니다.
Common KSA Profile — 담긴 Task들의 KSA 평균 Top 5. 채용 면접 기준 설정, 교육 우선순위 결정에 활용
KSA Distance Matrix — Task 간 36차원 KSA 코사인 유사도. 90%+ 녹색 = 교차교육 용이, 75% 미만 빨간색 = 별도 교육 필요
Training Gap Analysis — Task 간 교육시간 차이, 교육유형 비교, 적용 표준 갭 분석. 어떤 자격이 추가로 필요한지 한눈에 확인
스크롤 시 우측 하단 ↑ 버튼으로 Analysis 패널로 빠르게 이동할 수 있습니다.

5. 필드 설명

SOC 코드 — 미국 노동부 O*NET 표준직업분류 코드
Tech Complexity — IEEE/SAE 기반 기술 복잡도 (1-5)
TRIR (Total Recordable Incident Rate) — OSHA 기준 총 기록가능 재해율. 연간 정규직 100인당 업무상 부상·질병 발생 건수. BLS 산업 평균 제조업 TRIR ≈ 3.4
DART (Days Away, Restricted, Transferred) — 근무일 손실·제한·전환을 수반한 재해율. TRIR의 하위 지표로 심각도 반영. 낮을수록 안전한 직무
위험등급 — LOW / MEDIUM / HIGH / CRITICAL
KSA 점수 — O*NET 중요도 평가 기반 Knowledge·Skills·Abilities (1-5), LinkedIn 130만 포스트 NLP로 실증 보정

6. 데이터 출처 및 공신력

O*NET 30.0 — 미국 노동부(DOL) 고용훈련청 공식 직업 데이터베이스. 1,000+ 직종의 Knowledge·Skills·Abilities 36차원 중요도 점수(1-5)를 추출하여 각 Task의 KSA 프로파일 구성에 사용
ESCO v1.2 — 유럽연합 집행위원회(EC) 공식 직업·역량 분류체계. 2,942개 직업과 13,485개 역량을 표준화. 각 Task에 ESCO 직업코드를 매핑하여 국제 비교 가능성 확보
OSHA IIF / 300 Log — 미국 산업안전보건청(OSHA) 산업재해 기록. 사업장이 법적으로 보고하는 재해율 데이터에서 TRIR(총 기록가능 재해율)과 DART(근무일 손실 재해율)를 NAICS 산업별로 추출
DOL RAPIDS — 미국 노동부 등록도제 프로그램 시스템. 직종별 공인 훈련시간 기준을 추출하여 각 Task의 교육유형(OJT, Apprenticeship 등) 분류에 사용
Industry Standards — IATF/VDA/CQI/AIAG(자동차), AS9100/NADCAP(항공우주), IEC 62660/UN 38.3(배터리), 10 CFR/NQA-1(원자력) 등 국제인증 기준. 각 Task에 해당하는 적용 표준을 ISO 배지로 표시
LinkedIn 1.3M NLP — LinkedIn 채용공고 130만 건 중 78K 클린 포스트를 NLP로 분석. O*NET KSA 점수의 실무 타당성을 검증·보정하는 데 사용 (21개 CSV, 정제 후 적용)

1. Browsing the Task Inventory

This tool contains 191 Tasks mapped to O*NET SOC codes, OSHA hazard levels, and industry standards (IATF, VDA, CQI, AIAG, AS9100, IEC, etc.). It covers 6 industries in the U.S. Southeast Energy Belt (AL, TN, GA, SC, NC): Automotive (43), EV Battery (31), Solar (18), Aerospace (32), General Manufacturing (53), Nuclear (14).

Use the search bar to find tasks by keyword, SOC code, or standard name. Click ▶ Details on any card to reveal full task description, safety data, and KSA scores.

2. Multi-Select Filters

Both Industry and Department support multi-selection. Click to toggle on/off, click All to reset. For example, selecting Automotive + EV Battery simultaneously shows tasks from both industries.

3. JD Matcher (Job Description Matching)

In the JD Matcher area at the top, paste any job description text or enter keywords directly and the tool automatically extracts keywords to score all 191 tasks by relevance. The top 25 matches are displayed with matched keywords highlighted.

4. Cart & Analysis

Click + Add on any task card to add it to your Cart. When the Cart has tasks, an Analysis panel appears automatically between JD Matcher and the task list.
Common KSA Profile — Average Top 5 KSA scores across carted tasks. Use for setting interview criteria and training priorities
KSA Distance Matrix — 36-dimension KSA cosine similarity between tasks. 90%+ (green) = easy cross-training, below 75% (red) = separate training needed
Training Gap Analysis — Training hours comparison, certification requirements side-by-side, and standards gap matrix showing which tasks need which additional certifications
Use the ↑ floating button (bottom-right) to quickly scroll back to the Analysis panel.

5. Field Descriptions

SOC Code — U.S. DOL O*NET Standard Occupational Classification
Tech Complexity — IEEE/SAE-based technology complexity score (1-5)
TRIR (Total Recordable Incident Rate) — OSHA-mandated rate of all recordable workplace injuries and illnesses per 100 full-time equivalent employees per year. BLS manufacturing industry average ≈ 3.4
DART (Days Away, Restricted, Transferred) — Subset of TRIR measuring incidents resulting in lost workdays, restricted duty, or job transfer. Reflects injury severity — lower is safer
Hazard Level — LOW / MEDIUM / HIGH / CRITICAL
KSA Scores — O*NET importance ratings for Knowledge, Skills, Abilities (1-5 scale), validated with LinkedIn 1.3M post NLP analysis

6. Data Sources & Credibility

O*NET 30.0 — U.S. Department of Labor occupational database. Knowledge, Skills, and Abilities importance ratings (1-5 scale) were extracted for 1,000+ occupations to build each task's 36-dimension KSA profile
ESCO v1.2 — European Commission's classification of occupations, skills, and qualifications (2,942 occupations, 13,485 skills). Each task is mapped to an ESCO occupation code for international comparability
OSHA IIF / 300 Log — Occupational Safety & Health Administration injury/illness records. TRIR (Total Recordable Incident Rate) and DART (Days Away/Restricted/Transferred) rates were extracted by NAICS industry code
DOL RAPIDS — Registered Apprenticeship Partners Information Database System. Certified training hour standards by occupation were used to classify each task's training type (OJT, Apprenticeship, etc.)
Industry Standards — IATF/VDA/CQI/AIAG (Automotive), AS9100/NADCAP (Aerospace), IEC 62660/UN 38.3 (Battery), 10 CFR/NQA-1 (Nuclear). Applicable standards are shown as ISO badges on each task card
LinkedIn 1.3M NLP — 1.3M LinkedIn job postings were processed via NLP (78K clean posts, 21 CSVs). Used to validate and calibrate O*NET KSA scores against real-world job market demand
